Diabetes Prediction

  • UI: HTML, CSS, JavaScript
  • Web Server: Flask

A user-friendly web application designed to check whether a patient is affected by diabetes based on input data. The application allows users to input the following parameters:

  • No of Pregnancies
  • Glucose Level
  • Blood Pressure Level
  • Insulin Level
  • BMI
  • Age
The system uses machine learning models to predict whether the user is likely to have diabetes.
