Nepali Handwritten Text Recognition

  • UI: HTML, CSS, JavaScript
  • Web Server: Flask

This project focuses on recognizing Nepali handwritten and printed text using machine learning. It converts scanned or photographed Devanagari-script text into editable format using a CRNN (Convolutional Recurrent Neural Network) model. The system supports both online and offline handwriting recognition modes and is especially effective for digitizing written Nepali documents.
