Bagh-Chal - The Tiger and Goats Game

  • Project Type: Game Development
  • Platform: Python (Tkinter / Pygame)

Bagh-Chal is a traditional strategy board game from Nepal, played on a 5Ã—5 point grid resembling alquerque. Pieces are placed at line intersections, not inside the squares. Movement is governed by the lines connecting these points.

The game unfolds in two distinct phases. In the first phase, all goats are placed one by one while tigers can move. In the second phase, both tigers and goats can move. The tigers aim to capture five goats by jumping over them, similar to draughts, though capturing is not mandatory. Goats win if they block all tigers from moving.

Key Features

  • Traditional Logic: Accurate implementation of Bagh-Chal rules and win conditions
  • Turn-Based System: Two-phase game logic for placement and movement
  • Strategic AI: Optional mode to play against AI-controlled tiger or goat
  • Visual Feedback: Highlighted legal moves and captures
