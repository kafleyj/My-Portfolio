Proteomics Chatbot

  • Model Used: LLaMA 3.2:1b with RAG (Retrieval-Augmented Generation)
  • Database: UniProt (253M protein entries)

The Proteomics Chatbot is an AI-powered assistant that enables users to query complex protein-related data from the UniProt database using simple natural language. The system merges semantic search techniques with domain-specific models to boost the quality and specificity of biological answers.

Key Approaches

  • Vector Search: FAISS engine combined with SentenceTransformer embeddings
  • Domain Knowledge: Uses ProtBERT for handling protein sequences
  • Hybrid Retrieval: Combines semantic vector results with UniProt API data

Results & Observations

  • Hybrid method achieved the best performance in accuracy and adaptability
  • API-only failed to understand nuanced natural queries
  • ProtBERT significantly improved biological relevance in retrieval
